Browser Help

The browser shell runs as static files and never talks to a game server. It starts with in-memory viewer state; the IndexedDB status is a browser capability probe, not a claim that correlated vault operations are wired.

ActionDefault bindingScope

Map, POV, and Split use the same bindings as the native shell. In Split, click a pane to focus it; view-scoped keys and the wheel route only to that pane. Tab swaps Split focus when a view surface owns keyboard focus, or toggles Map/POV in a single-view presentation. POV look requires holding the primary button while dragging.

Feature Tiers

Low favors small caches, Mid expands the working set, and High permits the largest presentation density. The shipped browser starts on Low; selecting another tier emits a typed host request and never changes generated identities. Add ?tier=low, ?tier=mid, or ?tier=high to start a diagnostics run at an exact tier.

Storage

The adapter opens an IndexedDB probe and reports its status, but session, discovery, preserve, route, and atlas effects currently return a typed unavailable result and keep the viewer on its in-memory state. Generated terrain, organisms, and renderer output are never persisted.

Browser Requirements

WebGPU is optional. When available, the browser selects the WebGPU atlas presentation path; device loss or unsupported adapters fall back to the CPU-composed map without changing world state. The current worker module is a capability/lifecycle probe and the shared controller still ticks the inline executor; worker-pool selection remains typed scaffold state.

POV Mode

POV mode hosts the same shared 3D terrain renderer as the native shell on a WebGPU canvas. Hold the primary button and drag to look; pointer capture keeps that drag active outside the canvas. Move with WASD/arrows (Space/Shift fly up/down), and use the mouse wheel to adjust the fly or walk speed. If no WebGPU adapter or device is available — or the device is lost — the viewer returns to map mode with a warning.

Unsupported

Networking, accounts, hosted worlds, multiplayer, and authoritative GPU simulation are outside Phase 7.
