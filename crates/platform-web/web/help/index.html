Feature Tiers

WebLow favors small caches and inline execution, WebMid targets ordinary desktop browsers, and WebHigh enables larger visible windows and refinement where supported. Tiers change budgets and presentation only.

Storage

This milestone reports memory storage. IndexedDB vault persistence, import, export, reset, and reload recovery are Phase 7 storage milestones.

Browser Requirements

WebGPU is optional for the current static scaffold. Workers and shared memory are detected later; the inline path remains the deterministic fallback.

Unsupported

Networking, accounts, hosted worlds, multiplayer, and authoritative GPU simulation are outside Phase 7.
